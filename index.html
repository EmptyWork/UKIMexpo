Program Studi Informatika

Sebagai Program Studi baru di UKIM (Universitas Kristen Indonesia Maluku), kami tidak mau kalah dengan mahasiswa-mahasiswi dari Program Studi lain.

Apa itu Program Studi Informatika?

Program studi Informatika merupakan Program studi yang baru di bentuk pada Bulan Juni 2019 di Universitas Kristen Indonesia Maluku dan untuk saat ini memiliki 60 mahasiswa-mahasiswi pejuang Titel S.Kom, Prodi Informatika diharapkan menjadi Prodi yang bisa menghasilkan para calon professional yang bisa menguasai Application Developer dan Network Engineering
